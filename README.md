# 🍬 Gestão de Doces - PWA Profissional

Uma solução moderna, leve e eficiente para pequenos empreendedores e vendedores de doces. Esta aplicação funciona como um **PWA (Progressive Web App)**, o que significa que pode ser instalada no seu celular e funcionar como um app nativo, operando de forma **100% offline**.

---

## 🌟 Funcionalidades Principais

* **📈 Gestão de Vendas:** Registro de vendas com seleção de quantidade e cálculo automático de preço total.
* **👥 Controle de Clientes:** Cadastro por nome e departamento, com integração direta para contato via WhatsApp.
* **📦 Controle de Estoque:** Gestão de catálogo com preços e quantidades. O estoque é atualizado em tempo real a cada venda.
* **💰 Fluxo de Caixa:** Histórico detalhado de recebimentos com data, hora e opção de cancelamento.
* **🎨 Temas Personalizáveis:** Escolha entre 3 níveis visuais: **Claro**, **Médio** e **Escuro**.
* **💾 Segurança e Backup:** Dados salvos localmente no navegador (`LocalStorage`). Inclui central de backup para exportar e importar dados em formato `.json`.

---

## 📱 Como Instalar no Celular (PWA)

Após abrir o link da aplicação no seu navegador móvel:

### **Android (Google Chrome)**
1.  Aguarde alguns segundos na página.
2.  Um aviso de "Adicionar à tela inicial" poderá aparecer automaticamente.
3.  Se não aparecer, clique nos **três pontinhos** no canto superior direito.
4.  Selecione **"Instalar aplicativo"** ou **"Adicionar à tela inicial"**.

### **iOS / iPhone (Safari)**
1.  Clique no botão de **Compartilhar** (ícone quadrado com uma seta para cima no centro inferior).
2.  Role a lista para baixo e clique em **"Adicionar à Tela de Início"**.
3.  Confirme o nome e clique em **Adicionar**.

## 🛠️ Tecnologias Utilizadas

* **React 18:** Lógica de interface e estados.
* **Tailwind CSS:** Design responsivo e sistema de temas.
* **Lucide React:** Iconografia moderna.
* **Service Workers:** Cache e persistência offline.
* **LocalStorage API:** Armazenamento de dados sem necessidade de servidor.

## ⚠️ Avisos Importantes

1.  **Limpeza de Cache:** Como os dados ficam no navegador, se você limpar os dados de navegação ou formatar o celular, os dados serão apagados.
2.  **Backup:** Recomenda-se utilizar a função **"Salvar Backup"** nas configurações (ícone da engrenagem) pelo menos uma vez por semana. Guarde o arquivo gerado em um local seguro (Google Drive, E-mail ou WhatsApp).
